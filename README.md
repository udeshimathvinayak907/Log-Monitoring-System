# Log Monitoring & Alert System

## 📌 Project Overview
This project simulates a Security Operations Center (SOC) log monitoring system using Bash scripting.

It continuously monitors login activity, detects suspicious failed login attempts, and generates alerts.

---

## ⚙️ Features
- Real-time log monitoring using Bash
- Detection of repeated failed login attempts
- Identification of suspicious IP addresses
- Automated alert generation
- Continuous monitoring using loops

---

## 🛠️ Tools & Technologies
- Bash Scripting
- Linux Commands (grep, awk, sort, uniq)
- Log Analysis Techniques

---

## 📂 Project Structure
- `monitor.sh` → Main monitoring script
- `sample.log` → Sample log file
- `alerts.txt` → Generated alerts
- `README.md` → Project documentation

---

## 🚀 How to Run

```bash
chmod +x monitor.sh
./monitor.sh
```
